Unix Shell Tutorial

This is a beginner's tutorial for the Unix shell.

Example 1

Display something on the terminal:

$ echo "Hello World" # The selection will include the $ sign.

Example 2

Set a variable value and display it:

$ name="Alain" # The selection will not include the $ sign. This is better.
$ echo "name = $name"
